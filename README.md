# 五笔字型 Unicode CJK 超大字符集编码数据库

## 文本格式

```
第一列：Unicode 编码
第二列：Unicode 字符
第三列：86版五笔字型编码（缺少量，CJK-F编码错误，需校对）
第四列：98版五笔字型编码（缺CJK-F，需校对）
第五列：06版五笔字型编码（需依国标字形校对）
第六列：笔顺（仅CJK前20902字）
```
