# 五笔字型 Unicode CJK 超大字符集编码数据库

## 文本格式

```
第一列：Unicode 编码
第二列：Unicode 字符
第三列：86版五笔字型编码（需依国家标准字形校对）
第四列：98版五笔字型编码（CJK仅前20902字，缺CJK-F，需校对）
第五列：06版五笔字型编码（需依国家标准字形校对）
第六列：中国大陆笔顺（需依国家标准字形和规范笔顺校对）
第七列：中国台湾笔顺（与“CNS11643中文標準交換碼全字庫”数据同步）
```
